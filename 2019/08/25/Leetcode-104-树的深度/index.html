Leetcode -104 树的深度

Posted on 2019-08-25 | | Visitors: | 次

题目描述

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。
说明: 叶子节点是指没有子节点的节点。

思路分析

这道题实在求树的深度，个人感觉是一个挺有意义的题，上篇博文提到过，树的问题可以用递归调用解决（自己调用自己），所以对这道题尝试了递归调用：
① 先判断有没有子节点，即node.left && node.right ==null,如果为空，则返回1
② 如果只有一个左节点或右节点，则返回另外一个节点的最大深度，举个例子，如果node.left == null 并且node.right != null,则返回 helper(node.right)+1;
③ 如果左右节点都不为null,则返回max(helper(node.right)+1,helper(node.left)+1)
