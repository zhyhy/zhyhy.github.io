Leetcode -112 路径总和

No tag
3. LeetCode
Publish Date:   2019-09-02

题目描述

给定一个二叉树和一个目标和，判断该树中是否存在根节点到叶子节点的路径，这条路径上所有节点值相加等于目标和。
说明: 叶子节点是指没有子节点的节点。

例子

给定如下二叉树，以及目标和 sum = 22，

输出：true,因为存在目标和为 22 的根节点到叶子节点的路径 5->4->11->2

思路分析

利用递归解决问题，对树采用深度优先前序遍历，talk is cheap,show me the code

简单讲解一下代码。这个代码是我从discuss区学来的，可以很好的学习树的深度优先遍历，以及更好的理解树的递归。

-> 首先判断root是不是null，如果为null，则返回false（这个我之前没有写，后来在测试的过程中根据测试样例加上去的）
-> 然后判断是不是叶子节点，也就是一个path是否到头，如果到头，就判断sum和root.val值是否相同。如果sum == root.val，则return true，否则返回false；如果不是叶子节点，则继续执行下一步。
-> return hasPathSum(root.left,sum-root.val) || hasPathSum(root.right,sum-root.val)；由于上一步sum != root.val，则sum变成了sum-root.val,继续判断root.left和root.right是否满足条件。
递归真的好神奇，慢慢体会
